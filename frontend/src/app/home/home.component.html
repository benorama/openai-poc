Welcome to OpenAI Playground!

This is a proof of concept to test OpenAI beta APIs:
https://beta.openai.com/docs/introduction/overview

Text completion

The main API used is the completion endpoint to generate or manipulate text:

  • Guide: https://beta.openai.com/docs/guides/completion/introduction
  • API: https://beta.openai.com/docs/api-reference/completions

Note: by default, it uses text-davinci-003 model, but this can be easily change in backend config.

Moderation

It also uses the moderation endpoint, that should be used before any completion endpoint request to comply with OpenAI terms of service:

  • Guide: https://beta.openai.com/docs/guides/moderation
  • API: https://beta.openai.com/docs/api-reference/moderations

Note: by default, it uses text-moderation-stable model, but this can be easily change in backend config.
